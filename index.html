    Updating Packages

    Package Control Packages are automatically updated when you enable the plugin. Git Packages can be updated using update-submodules.sh which will bring each repository up-to-date with the master branch of the origin repository.

    Enable Package Control

    I've disabled the Package Control plugin by default because it slows down Sublime. Just open up Packages/User/Preferences.sublime-settings, comment out the "Package Control" line and save. Sublime will load it automatically.
